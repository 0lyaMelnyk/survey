## How to work with Backend:
The easiest way is to use VS and the IIS Express server.
### Backend Tools
* Visual Studio 2022
* .NET 6
* IIS Express
 
##  How to up Frontend:
  1. Install Node.js any version
  2. Run further commands in the terminal:
`npm install`;
`npm start`
### Frontend Tools
* Visual Studio Code
* node.js v14.17.5
* npm 6.14.14
